# Assets directory

Place brand and layout imagery here so the HTML references resolve:
- `logo/` for LaunchR icons and wordmarks (e.g., `icon-logo-dark.png`, `icon-logo-transparent.png`, `launchr-lockup.png`).
- `hero/` for optional hero background art (e.g., `launchr-hero-bg.png`).
- `screenshots/` for sample layout thumbnails (e.g., `saas.png`, `corporate.png`, `local.png`, `direct-response.png`).

Empty placeholder files are tracked with `.gitkeep` in each folder until assets are added.
